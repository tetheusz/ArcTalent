# Project: Archetypes

Reputation operating system for the Arc ecosystem. Transitioning from a talent board to a gamified identity layer where contributors earn XP and levels in specific Archetype classes.

## Objectives
- Build a premium, cinematic identity layer for builders.
- Implement a verifiable on-chain reputation system using SBTs.
- Create a gamified "Character Sheet" profile experience.

## Tech Stack
- Next.js 15, TypeScript, Drizzle ORM, SQLite, Wagmi, Three.js.
