# Conventions

- **Visual Identity**: "Cockpit" aesthetic — sharp edges (0px radius), high-contrast dark theme, transparent glass surfaces, cyan/purple accents.
- **Typography**: High tracking (letter-spacing) for labels, uppercase for CTAs.
- **Data Flow**: Use Server Actions for all mutations; use Drizzle queries for data fetching in server components.
- **State Management**: Minimal client-side state; rely on URL state and server revalidation.
- **Naming**: 5-class Archetype system (Developer, Sentinel, Creator, Scholar, Strategist).
