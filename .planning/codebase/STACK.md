# Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS Modules (Sharp-edge aesthetic)
- **Database**: SQLite with Drizzle ORM
- **Web3**: Wagmi, Viem, ConnectKit
- **3D/Graphics**: Three.js, React Three Fiber
- **Icons**: Lucide React
- **Blockchain**: Arc Network (L2 Testnet)
